# sysinfo
a command to show the system info as full as possible

## usage: 

```bash
➜  sysinfo git:(master) ✗ ./sysinfo -h
Usage of ./sysinfo:
  -ditto string
    	ditto mark (same as above (default "\"")
  -format string
    	display format json/table (default "table")
```
